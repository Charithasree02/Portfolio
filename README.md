# Charitha Sree — Portfolio

A professional portfolio website built with React + Vite + TypeScript + TailwindCSS.

## Tech Stack
- React 18 + TypeScript
- Vite
- TailwindCSS v4 + Shadcn UI
- Framer Motion (animations)
- Wouter (routing)

## Sections
- Hero
- About
- Projects
- Skills
- Achievements & Awards
- Contact

## Getting Started

```bash
npm install
npm run dev
```
